## Jak uruchomić projekt?

# Frontend

````
npm install
npm run dev
````

# Backend
````
cd server
nodemon index.js
````
## Technologie

# Frontend: React, Typescript

# Backend: Node.js, Express, MongoDB

# Autentykacja: JWT (JSON Web Tokens)

# Inne narzędzia: Postman, Git, GitHub
