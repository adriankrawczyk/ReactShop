# How to Run the Project?

## Frontend

```
npm install
npm run dev
```

## Backend

```
cd server
nodemon index.js
```

## Technologies

### Frontend: React, TypeScript

### Backend: Node.js, Express, MongoDB

### Authentication: JWT (JSON Web Tokens)

### Other Tools: Postman, Git, GitHub
